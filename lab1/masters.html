Цікаві факти

Скульптори криги використовують спеціальні інструменти, щоб запобігти розтріскуванню матеріалу.

Тату-майстри для тварин створюють фарби, що повністю змиваються водою.

  1. Скульптор криги — створює витвори мистецтва з льоду.
  2. Тату-майстер для тварин — прикрашає хутро фарбою, без шкоди для тварини.
  3. Майстер годинників — ремонтує старовинні механізми.
